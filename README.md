# Django Blog App

A user-friendly blogging platform designed to simplify content creation and management.

## 📌 Overview
This project focuses on creating a clean and intuitive blogging experience for users who want to publish content without technical complexity.

## 👤 Target Users
- Aspiring writers
- Content creators
- Users looking for a simple blogging platform

## 💡 Problem
Many blogging platforms are either too complex or lack flexibility, making it difficult for users to focus on writing.

## 🧠 Key Insights
- Users prefer simplicity over feature-heavy interfaces  
- Easy navigation improves engagement  
- Quick content creation is a priority  

## 🚀 Solution
Designed a streamlined blogging platform that allows users to easily create, edit, and manage posts with minimal friction.

## ✨ Features
- User authentication (login/signup)
- Create, edit, delete blog posts
- Clean and minimal UI
- Structured content display

## 🛠 Tech Stack
- Django
- HTML, CSS
- SQLite
